Components

Component Description Cost
XIAO ESP32C6 Microcontroller with BLE ~$5
HX711 Load cell amplifier ~$2
Load Cell 4-pin strain gauge (50-200kg) ~$10
OLED Display 128x32 I2C mini display ~$3
Total ~$20

Wiring

HX711 → ESP32C6

  • VCC → 3.3V
  • GND → GND
  • DT → D4
  • SCK → D5

OLED → ESP32C6

  • VCC → 3.3V
  • GND → GND
  • SCL → D0
  • SDA → D1
